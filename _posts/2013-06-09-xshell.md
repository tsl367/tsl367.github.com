#shell使用指南
##ZMODEM功能
    yum install lrzsz
    rz 上传
    sz 下载
##快捷键
    alt + o 打开终端
    alt 1-9 切换
    ctrl + alt 切换
    ctrl + shift + n 打开新选项卡
##vim的小键盘不能使用的问题
在会话的属性中，将VT模式的初始数字键盘设置为普通
##配色方案
1. 保存成xcs文件，导入；
2. 在每个会话窗口设置成默认
